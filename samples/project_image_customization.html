Project Images Customization Guide

Note: Replace the image URLs below with your actual image paths. You can use local images (e.g., "images/project1.jpg") or external URLs.

Method 1: Using IMG Tag (Recommended)

<!-- HTML --> <div class="project-header-img"> <img src="https://via.placeholder.com/400x200/667eea/ffffff?text=AI+Project" alt="AI Project" class="project-image"> </div> <!-- CSS --> .project-header-img { height: 200px; overflow: hidden; } .project-image { width: 100%; height: 100%; object-fit: cover; transition: transform 0.3s ease; } .project-card:hover .project-image { transform: scale(1.05); }

Method 2: Using Background Image

<!-- HTML --> <div class="project-header-bg" style="background-image: url('https://via.placeholder.com/400x200/764ba2/ffffff?text=Mobile+App');"></div> <!-- CSS --> .project-header-bg { height: 200px; background-size: cover; background-position: center; background-repeat: no-repeat; }

Method 3: Background Image with Hover Overlay

<!-- HTML --> <div class="project-header-overlay" style="background-image: url('https://via.placeholder.com/400x200/4ecdc4/ffffff?text=Game+Project');"> <div class="project-overlay">View Project</div> </div> <!-- CSS --> .project-header-overlay { height: 200px; background-size: cover; background-position: center; position: relative; } .project-overlay { position: absolute; top: 0; left: 0; right: 0; bottom: 0; background: rgba(102, 126, 234, 0.7); display: flex; align-items: center; justify-content: center; color: white; font-size: 2rem; opacity: 0; transition: opacity 0.3s ease; } .project-card:hover .project-overlay { opacity: 1; }
